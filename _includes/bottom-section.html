ZENv is a coding environment manager for programmers.

It allows you to keep your shell environment in source control, thereby keeping everyone in sync and allowing access to shared dev tools, aliases, and more.
Terminal

my_terminal$ cd db_backend

my_terminal$ phing -logger -Dprops=../myprops

...

 

 

my_terminal$ cd ../app_frontend

my_terminal$ make

...

The Old Way

Each project has a unique set of commands to build which require developers to understand each build system and those respective capabilities.

ZENv

ZENv$ use db_backend

ZENv$ build

...

 

 

ZENv$ use app_frontend

ZENv$ build

...

Add ZENv to your Dev Environment

With ZENv, there's no need to spend time figuring out the subtleties of each project's build system. Just build!
